# Contributing to Sonic Ferrocrete

Thanks for your interest! This repo explores magneto-acoustic forming of gypsum/cement mixes.

## How to contribute
1. **Experiments:** Open an *Experiment Run* issue (template provided) and add photos to `experiments/results_gallery/`.
2. **Hardware:** For `hardware/*.scad`, include a photo/screenshot of a print or render and the parameters changed.
3. **Docs/Data:** Append rows to the CSV templates in `data/`. Do not commit raw video; link it in issues instead.

## Style
- Keep files small and readable. Prefer parameterized CAD.
- Use descriptive commit messages (e.g., `hardware: add 200mm tray variant`).

## Safety
- Use hearing protection at sustained SPL.
- Avoid inhaling dust (mask when handling powders).
- Neodymium magnets can pinch; keep away from cards/drives.

## Getting help
Open a GitHub issue with label `question` and a short description.
